Introduction

Installing and Usage

Installing qdb is easy, all you need is some webspace and a mysql server. Most shared and or free hosting support this.

Firstly create a database, you will need the following info:

  • MySQL host
  • MySQL user
  • MySQL password
  • MySQL database

If you have no clue what either of those things are: click here.

So you've got your database set up? Great! Let us open phpMyAdmin and run the .sql file we have. You can do this by either; copying the code; manually creating the tables, or uploading the file and letting phpMyAdmin do the work for you.

Next stop: conf.inc.php

Change the variables to your desire, save the file and re-upload.

Enjoy your short URL service!
